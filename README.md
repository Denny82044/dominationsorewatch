# Domination Score Watch
A simple python script that uses Dolphin Memory Engine to print the current score of port 1 in Mario Party 4's Domination

# Installation Guide
## Requirements
1. Python 3.10+

2. Everything inside of requirements.txt
```
pip install -r requirements.txt
```
## Downloading & running the program
1. Check the releases tab on this repo, and download the latest release for the region you want. (https://github.com/Denny82044/dominationsorewatch/releases)
2. Extract the .zip when downloaded.
3. Run the main.py file.
4. Load Dolphin, then run Mario Party 4.
5. Load into Domination, and the program will show you port 1's current score in the console and in a tkinter window.

# Notes/Important info
1. The console will be very spammy, this is intended.
2. If not in Domination, the program will show random values, this is normal.
3. If there are any bugs, report them on the repo or to me directly on Discord (denny82044)
